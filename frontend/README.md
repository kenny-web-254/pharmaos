# NexaCore Frontend

Premium React PWA for NexaCore business OS.

## Quick Start

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
# http://localhost:5173
```

### Build
```bash
npm run build
```

### Preview
```bash
npm run preview
```

## Project Structure

```
src/
├── components/     # React components
│   └── layouts/    # Layout components
├── pages/          # Page components
├── store/          # Zustand state
```
